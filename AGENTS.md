# Agent Instructions

## Development Server

To start the local development server, run:

```bash
// turbo
npx -y serve -l 3000
```

This will serve the static site at `http://localhost:3000`.

## Project Structure

- `index.html` - Main entry point
- `styles/v2/main.css` - Primary stylesheet (Event Horizon theme)
- `scripts/v2/` - ES Module JavaScript files
- `assets/` - Static assets (images, models, etc.)

## Tech Stack

- **HTML5** + **Tailwind CSS v4** (CDN)
- **Vanilla CSS** for custom styling
- **ES Modules** for JavaScript
- **Three.js** for 3D rendering
- **GSAP** + **Lenis** for animations
